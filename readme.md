# JX3BOX

公共基础函数模块，路径常量，外部通用接口地址，全局主题等

## Step.1 Install

```
$ npm install @jx3box/jx3box-common --save
```

## Step.2 Import

```javascript
import { JX3BOX } from "@jx3box/jx3box-common";
import User from "@jx3box/jx3box-common/js/user.js";
```
