# JX3BOX
公共基础函数模块，路径常量，外部通用接口地址，全局主题等

## Step.1 Install
```
npm install @jx3box/jx3box-common
```

## Step.2 Import
```javascript
import {JX3BOX , Utils} from '@jx3box/jx3box-common';
```
